# Customizable Planner

## Installation
### Prerequisite
*  node.js, vue, electron 의 설치가 필요합니다.
```c
사용자 Installation 하는 방법
> npm install -g vue-cli
> npm install -g electron-forge

```
### 실행
```c
yarn electron:serve
```
